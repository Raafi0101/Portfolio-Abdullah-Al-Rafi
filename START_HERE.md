## 🆘 Need Help?

| Problem | Solution |
|---------|----------|
| Port 3000 busy | `npm run dev` uses port 3000, change to `PORT=3001 npm run dev` |
| Module errors | Delete `node_modules` + `npm install` again |
| Images not showing | Put them in `public/` folder (not `src/`) |
| Build fails | Check terminal error, likely missing `npm install` |
| No hot reload | Make sure you're running `npm run dev` (not `npm start`) |
